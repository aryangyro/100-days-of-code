### Guessing Game
A simple number guessing game where the computer picks a random number and the user tries to guess it.

The game:

- Gives limited chances based on difficulty
- Tells the user whether to guess higher or lower
- Ends when the number is guessed or chances run out

This helped me understand how game flow works using loops and conditions.
